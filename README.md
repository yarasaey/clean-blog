 📝 Clean Blog - Content Management System

 📌 Description

Clean Blog is a simple and elegant Content Management System (CMS) built with Laravel.
It allows users to read blog posts and interact with content, while providing administrators with full control over managing posts, categories, and authors.


 💡 Business Idea

The idea behind Clean Blog is to create a lightweight blogging platform where:

* Users can explore and read articles 
* Content is organized by categories and authors
* Admins can manage all blog content dynamically


🎯 Project Goals

* Build a dynamic CMS using Laravel
* Implement Authentication & Authorization
* Provide a clean and user-friendly UI
* Manage blog content efficiently

 ⚙️ Installation

1. Clone the repository


git clone https://github.com/yarasaey/clean-blog.git
cd clean-blog
2. Install dependencies


composer install
npm install
3. Setup environment

cp .env.example .env
php artisan key:generate


4. Configure database in `.env`

env
DB_DATABASE=your_db_name
DB_USERNAME=root
DB_PASSWORD=


5. Run migrations

php artisan migrate

6. Run the project

php artisan serve
